# ULTRA FRAME

Open source hardware and design files for a **leverless controller**— arcade-style input without a joystick gate, built for fighting games and anyone who prefers all-button layouts.

This repository is **work in progress**. KiCad projects, mechanical CAD, firmware, and documentation will land here as the design matures. Watch this space or star the repo if you want to follow along.
